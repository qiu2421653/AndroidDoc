字符串中某几个字符颜色变换 

mBtnVerify.setText(millisUntilFinished / 1000 + "秒后\n重新发送");// 设置倒计时时间 // 设置按钮为灰色，这时是不能点击的 Spannable span = new SpannableString(mBtnVerify.getText() .toString());// 获取按钮的文字 span.setSpan(new ForegroundColorSpan(Color.RED), 0, 2, Spannable.SPAN_INCLUSIVE_EXCLUSIVE);// 讲倒计时时间显示为红色 mBtnVerify.setText(span);
