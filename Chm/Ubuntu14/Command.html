Command not found  

新安装的64位ubuntu server下sudo:add-apt-repository:command not found 问题 比如想要在Ubuntu Server 安装个oracle-jdk7环境， 直接执行命令： sudo add-apt-repository ppa:webupd8team/java 错误来啦：sudo: add-apt-repository:command not found 网上解决办法是直接安装工具包 命令：sudo apt-get install python-software-properties 亲测安装后还是报command not found，所以依赖包还没有安装完全，少了什么呢？ 除了要安装python-software-properties外还需要software-properties-common ok,执行安装命令：sudo apt-get install software-properties-common
